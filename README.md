# Viora Pulse 🇰🇪
### Know Before You Go

Real-time crowd-sourced utility status platform for Kenya.

## What it does
Check water and electricity status in your area before leaving home.

## Tech Stack
- **Backend:** FastAPI + PostgreSQL
- **Frontend:** React + Leaflet (OpenStreetMap)
- **Deployment:** Docker + Render.com

## MVP Features
- Report water/electricity availability by location
- View live status on a map
- Confidence scoring based on number of reports
